Pourquoi Git ?

J'utilise git depuis + de 5 ans et vraiment .. c'est pratique

Le travail à plusieurs est vraiment très simple, l'historique permet d'avoir toujours une copie du travail passé car casser la prod ca reste simple à faire

Le versionning

Le versionning étant absolument indispensable aujourd'hui, Git effectue de manière très propre son travail, associé à Gitlab (pour les projets privé :) et Github pour les projets public c'est parfait.
